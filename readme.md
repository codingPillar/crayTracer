# crayTracer

## Simple CPU side ray tracer

## Build (Multiple possible ways)
```
cmake -S . -B build
cd build && make
```
```
make cmake
```
```
make tracer
```

### Web version is already deployed on pages at: https://codingpillar.github.io/crayTracer/
